Communities

Different communities support Soft-Shake for its first edition. We thank them.

 

GenevaJUG is a non-profit organization aimed to promote knowledge, new technologies and best practices of the Java ecosystem in the Geneva Area.

We meet on the last Tuesday of each month. The meetings are free and open to all and consist of technical presentations, demonstrations of products or round table discussions. After the meeting we meet over a drink and a mini-buffet.

Contact: team@genevajug.ch

 

 

Duchess France is a virtual french Java User Group who brings together Java developers to promote woman it IT. We are a virtual community but also have some local group, which work in collaboration with others french JUG (Paris JUG, Lyon JUG etc.). We organise some events, open to all, like a work group to pass the SJCP certification or informal chat before others JUG meeting

Contact: duchessfr@gmail.com

 

 

AlpesJug is the Java User Group of Grenoble (France) which aims to be a place where Java users (professionals, teachers, students, ...) can meet and talk about Java related technologies. We meet usually the last monday of each month. The meetings are free and open to all and
consist of technical presentations, demonstrations of products or round table discussions. After the meeting we meet over a drink and a mini-buffet.

Contact: Emmanuel Hugonnet

 

Lorraine JUG aims to gather users of Java technologies in Lorraine (eastern region of France). Hosted at ESIAL in Nancy, we want to create bridges between IT professionals and students by sharing experiences around Java.
Lorraine JUG organizes about 6 events a year. If you are interested in speaking at Lorraine JUG, please send an e-mail to info@lorrainejug.org.

 

The Paris JUG is a french Java User Group who brings together Java developers to share ideas and discuss advances in technology. We meet on the second Thursday of each month (except August). The meetings are open to all, and usually consist of one presentation from developers, open source community, vendors, or academics. Afterwards we usually have an informal chat and drink to socialise.

Contact: Antonio Goncalves
