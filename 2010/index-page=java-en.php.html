Java

The Java track is supported by the GenevaJUG

 

The GenevaJUG is a Java User Group aimed to bring together actors of the language (professional, open source communities, institutions, students, teachers, ...) to exchange ideas and discuss the technological advances of the Java platform.

 

The GenevaJUG community increases over the various sessions, bringing together up to 150 people around the same topic.

 

The programming language Java is one of the most used programming language and has a large Open Source community.

 

This track is intended, as the JUG, speaking on existing Java technologies, the future of languages running on the JVM, talking about best-practices and to open minds about new technologies available on the platform.
